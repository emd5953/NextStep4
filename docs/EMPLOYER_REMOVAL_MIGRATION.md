## Historical Context

This document was created during the platform simplification process when NextStep transitioned from a dual-sided platform (job seekers + employers) to a streamlined, job-seeker-focused application.

### What Was Removed (January 2026)

- Employer dashboard and management features
- Job posting functionality (replaced with external API jobs)
- Employer-applicant messaging system
- Company profile management
- Application status management by employers

### Current Platform (2026)

NextStep is now a **job seeker platform** that:
- Sources real jobs from JSearch API
- Provides AI-powered job matching
- Offers swipe-based job discovery
- Includes RAG-powered AI chatbot for help
- Tracks user applications
